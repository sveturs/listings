## Операционная модель

### Схема работы платежей
1. **Покупатель** → оплачивает заказ
2. **Sve Tu** → удерживает средства (escrow)
3. **Доставка** → подтверждение получения
4. **Продавец** → получает выплату (минус комиссия)

### Комиссионная структура
- **Входящие платежи**: 
  - Карты: 1.5-2.5% (зависит от типа)
  - Банковский перевод: 0.3%
- **Выплаты продавцам**:
  - Массовые выплаты через API
  - Еженедельные или по запросу

### Валютные операции
- **Основная валюта**: RSD
- **Конвертация**: EUR, USD через банк
- **Региональные валюты**: 
  - BAM (Босния)
  - MKD (Македония)
  - HRK → EUR (Хорватия)
