## Цель интеграции

Наша платформа Sve Tu является одним из ведущих маркетплейсов в Сербии. Мы хотим интегрировать услуги Post Express для предоставления нашим клиентам следующих возможностей:

1. **Автоматический расчёт стоимости доставки** в режиме реального времени
2. **Выбор способа доставки:**
   - Курьерская доставка на адрес (PE_Danas_za_sutra_19)
   - Доставка в пункты выдачи (PeBox локации)
   - Доставка в почтовые отделения
3. **Отслеживание отправлений** в личном кабинете покупателя
4. **Массовое создание отправлений** для продавцов
5. **Автоматическая печать этикеток** и накладных

## Техническая готовность

Мы уже полностью реализовали интеграцию с WSP API согласно вашей документации:

### Backend (Go):
- ✅ Клиент для WSP API с поддержкой всех транзакций
- ✅ Сервисный слой для всех операций
- ✅ REST API endpoints для frontend
- ✅ Модели данных и обработка ошибок
- ✅ Кеширование и оптимизация

### Frontend (React/TypeScript):
- ✅ Компонент выбора способа доставки (3 шага)
- ✅ Интеграция с корзиной покупок
- ✅ Отслеживание статуса доставки
- ✅ Поддержка мультиязычности (SR/RU/EN)

### Реализованные транзакции WSP API:
- ID=3: GetNaselje (поиск населённых пунктов)
- ID=4: GetUlica (поиск улиц)
- ID=6: ProveraAdrese (проверка адреса)
- ID=9: Provera dostupnosti usluge
- ID=11: Poštarina pošiljke (расчёт стоимости)
- ID=63: Pojedinačno praćenje (отслеживание)
