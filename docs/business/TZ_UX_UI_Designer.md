# Техническое задание для UX/UI дизайнера платформы Sve Tu

## Общее описание проекта

**Sve Tu** — революционная маркетплейс-платформа в Сербии, объединяющая функционал C2C и B2C торговли с передовыми технологиями. Платформа позволяет частным лицам размещать объявления, а бизнесам создавать полноценные витрины (storefronts) с расширенными возможностями.

## Цель проекта

Создать интуитивно понятный, современный и конверсионный дизайн для маркетплейса, который обеспечит:
- Легкое размещение и поиск товаров/услуг
- Удобное управление витринами для бизнеса
- Безопасные сделки с эскроу-защитой
- Эффективную коммуникацию между продавцами и покупателями

## Основные функции платформы

### 1. Поисковая система
- **OpenSearch** с нечетким поиском (fuzzy search)
- Фильтры по категориям, атрибутам, цене, местоположению
- Поиск на карте с кластеризацией
- Сохраненные поиски и уведомления

### 2. Категории и атрибуты
- Иерархическая система категорий
- Динамические атрибуты для каждой категории
- Умные фильтры на основе атрибутов
- Визуальная навигация по категориям

### 3. Создание объявлений
- **AI-powered создание** (анализ фото, генерация описания)
- **Smart создание** с подсказками
- Пошаговый wizard для новичков
- Bulk upload для профессионалов

### 4. Витрины (Storefronts)
- Персонализированный дизайн витрины
- Dashboard с аналитикой и метриками
- Управление товарами и заказами
- Интеграция с внешними системами (XML/CSV/API)
- Отслеживание остатков на складах

### 5. Платежи и логистика
- **Эскроу для B2C** (безопасные онлайн-платежи)
- **Наложенный платеж** для C2C и B2C
- Интеграция с курьерскими службами
- Отслеживание доставки

### 6. Коммуникация
- Встроенный чат с анимированными эмодзи
- Уведомления в реальном времени
- Шаблоны сообщений для продавцов
- История переписки

### 7. Отзывы и рейтинги
- Система рейтингов для продавцов
- Отзывы с фото и видео
- Верификация покупок
- Модерация отзывов

### 8. Специальные функции
- **Автоматические скидки** (Черная пятница, сезонные)
- **Картографический сервис** с приватностью адресов
- **Мультиязычность** (сербский, английский, русский)
- **PWA** поддержка для мобильных

## Требования к дизайну

### 1. Общие принципы
- **Mobile-first подход** (60% трафика с мобильных)
- **Доступность** (WCAG 2.1 Level AA)
- **Скорость загрузки** (оптимизация изображений)
- **Интуитивность** (минимум обучения)

### 2. Визуальный стиль
- Современный, чистый дизайн
- Использование DaisyUI компонентов как основы
- Поддержка светлой/темной темы
- Акцентные цвета для CTA элементов

### 3. Ключевые страницы для дизайна

#### Главная страница
- Hero-блок с поиском
- Популярные категории (визуальная сетка)
- Рекомендованные товары
- Карта с товарами поблизости
- Статистика платформы
- Блок преимуществ

#### Страница поиска/каталога
- Умные фильтры (адаптивные под категорию)
- Переключение вид: сетка/список/карта
- Быстрый просмотр товара
- Бесконечная прокрутка
- Сохранение поисков

#### Карточка товара
- Галерея изображений с zoom
- Ключевая информация above the fold
- Динамические характеристики
- Блок продавца с рейтингом
- Похожие товары
- Кнопки действий (купить, чат, поделиться)

#### Создание объявления
- 3 режима: AI, Smart, Manual
- Прогресс-бар для wizard
- Drag&drop для фото
- Предпросмотр объявления
- Подсказки и примеры

#### Dashboard витрины
- Ключевые метрики (карточки)
- Графики продаж и посещений
- Таблица заказов с фильтрами
- Уведомления о низких остатках
- Быстрые действия

#### Страница витрины
- Брендированный header (лого, баннер)
- Навигация по категориям витрины
- Сетка товаров с фильтрами
- Информация о магазине
- Отзывы и рейтинг

#### Чат
- Список диалогов с превью
- Окно переписки с историей
- Быстрые ответы/шаблоны
- Отправка изображений
- Индикаторы онлайн/печатает

#### Checkout (для B2C)
- Пошаговый процесс
- Выбор способа оплаты
- Выбор доставки с расчетом
- Эскроу-информация
- Подтверждение заказа

### 4. Адаптивность

#### Mobile (320-768px)
- Упрощенная навигация (burger menu)
- Вертикальная ориентация элементов
- Свайпы для галерей
- Фиксированные CTA кнопки
- Оптимизация форм ввода

#### Tablet (768-1024px)
- 2-колоночные layouts
- Боковые панели как модалки
- Адаптивные таблицы

#### Desktop (1024px+)
- Полнофункциональный интерфейс
- Множественные колонки
- Расширенные фильтры
- Hover эффекты

### 5. Компоненты для разработки

Необходимо создать дизайн-систему включающую:

#### Базовые элементы
- Типографика (заголовки, текст, ссылки)
- Цветовая палитра
- Иконки (набор 100+ иконок)
- Кнопки (primary, secondary, ghost, sizes)
- Формы (inputs, selects, checkboxes, radios)
- Карточки товаров (разные размеры)
- Модальные окна
- Тосты и уведомления
- Лоадеры и скелетоны

#### Сложные компоненты
- Header с поиском
- Footer с навигацией
- Фильтры (desktop/mobile)
- Галерея изображений
- Карта с маркерами
- Графики и диаграммы
- Таблицы с сортировкой
- Пагинация
- Breadcrumbs

### 6. Особые требования

#### SEO-оптимизация
- Правильная иерархия заголовков
- Alt-тексты для изображений
- Микроразметка для товаров
- Человекопонятные URL

#### Производительность
- Lazy loading изображений
- Оптимальные размеры изображений
- Минимизация анимаций на mobile
- Progressive enhancement

#### Безопасность
- Визуальные индикаторы безопасности
- Четкие warning для важных действий
- Защита от случайных кликов
