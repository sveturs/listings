# Промпт для 2D логотипа "Sve Tu" - одна грань кубика Рубика

## Финальный оптимизированный промпт

```
Create a 2D flat logo for "Sve Tu" marketplace featuring a single Rubik's cube face with 3x3 grid (9 squares total). Design specifications: perfect square format, clean geometric grid with thin white or light gray dividing lines between squares, each square MUST contain a DIFFERENT and UNIQUE minimalist marketplace icon in flat design style - NO REPEATED ICONS, ALL 9 ICONS MUST BE DISTINCT AND DIFFERENT FROM EACH OTHER. 

Icons arrangement - MANDATORY, ALL 9 MUST BE PRESENT AND UNIQUE (left to right, top to bottom):
1. House/home icon (real estate) - ONLY ONE house icon
2. Car silhouette (automotive) - ONLY ONE car icon
3. Shopping bag (retail) - ONLY ONE shopping bag
4. Briefcase (jobs/career) - ONLY ONE briefcase
5. Location pin (local services) - ONLY ONE location pin
6. Wrench/tools (repairs/services) - ONLY ONE wrench/tool
7. T-shirt/clothing hanger (fashion) - ONLY ONE clothing item
8. Smartphone (electronics) - ONLY ONE phone
9. Price tag or handshake (deals/business) - ONLY ONE price tag or handshake

CRITICAL REQUIREMENT: Each of the 9 squares must have a completely different icon - no duplicates, no repeated symbols, exactly 9 unique icons total

Color scheme: Each square has a different background color creating a cohesive but diverse palette - blue (#2563EB), green (#10B981), orange (#F59E0B), purple (#8B5CF6), red (#EF4444), cyan (#06B6D4), yellow (#FCD34D), pink (#EC4899), indigo (#6366F1). Icons are white or light colored for contrast against colored backgrounds.

Style requirements: Ultra-flat design with no shadows or 3D effects, perfectly aligned grid structure, all icons same visual weight and simplified to essential shapes, modern minimalist aesthetic, sharp corners for digital use, scalable vector style, professional but approachable, represents marketplace diversity through color variety while maintaining design unity through consistent icon style and grid structure.

Additional details: Optional subtle rounded corners on outer square border (2-4px radius), thin gaps between squares (1-2px) filled with white or transparent, ensuring logo works on both light and dark backgrounds, memorable pattern that suggests variety and completeness, avoiding complexity while conveying "everything here" concept through the nine different category representations.
```
