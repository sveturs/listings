### Специфичные атрибуты по категориям:

#### Электроника:
- screen_size (размер экрана)

#### Мода:
- size (размер)
- material (материал)
- gender (пол)
- season (сезон)

#### Недвижимость:
- heating_type (тип отопления)
- construction_year (год постройки)
- elevator (лифт)
- security (охрана)

#### Автомобили:
- engine_size (объем двигателя)
- doors (количество дверей)
- seats (количество мест)
- drive_type (привод)

#### Услуги:
- price_type (тип цены: за час/проект/месяц)
- certification (сертификаты)
- language (языки)
- portfolio_url (портфолио)
