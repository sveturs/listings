# Команда и операции Sve Tu Platforma

## Философия построения команды

### Основные принципы
1. **Hire Slow, Fire Fast** - тщательный отбор, быстрые решения
2. **Culture Fit > Skills** - культура важнее навыков
3. **Remote First** - распределенная команда с первого дня
4. **Ownership Mentality** - каждый думает как основатель
5. **Continuous Learning** - постоянное развитие

### Корпоративная культура

#### Ценности
- **Пользователь превыше всего** - все решения через призму пользы
- **Скорость и качество** - быстро, но не в ущерб качеству
- **Прозрачность** - открытая коммуникация на всех уровнях
- **Результат важнее процесса** - фокус на достижениях
- **Командная игра** - успех команды = личный успех

#### Принципы работы
```
DO:
✓ Принимай решения быстро
✓ Экспериментируй и учись
✓ Говори прямо и честно
✓ Бери ответственность
✓ Помогай коллегам

DON'T:
✗ Не усложняй без необходимости
✗ Не жди идеального момента
✗ Не скрывай проблемы
✗ Не работай в силосах
✗ Не бойся ошибаться
```

## Найм и onboarding

### Процесс найма

```
Этапы отбора:
1. Application review (1-2 дня)
   - CV screening
   - Portfolio/GitHub check

2. HR Screen (30 мин)
   - Culture fit
   - Motivation
   - Expectations

3. Technical Interview (1-2 часа)
   - Coding challenge
   - System design
   - Problem solving

4. Team Fit (45 мин)
   - Meet potential colleagues
   - Q&A session

5. Founder Chat (30 мин)
   - Final culture check
   - Offer discussion

Timeline: 2-3 недели max
```

Day 1:

Day 4-5:

- Buddy meetings
- Feedback session

#### Week 2-4: Ramp-up
