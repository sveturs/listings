Тест AdminGuard через Redux Store

⚠️ Важно: Этот тест нужно выполнять непосредственно на странице React приложения, а не в отдельном HTML файле.

Шаг 1: Откройте приложение

Перейдите на http://localhost:3001

Шаг 2: Откройте консоль разработчика (F12)

Нажмите F12 или Ctrl+Shift+I для открытия DevTools

Шаг 3: Скопируйте и выполните этот код в консоли

Код для модификации пользователя через Redux:

// Получаем Redux store через window.__REDUX_STORE__ // (это доступно благодаря StoreProvider) const store = window.__REDUX_STORE__; if (!store) { console.error('Redux store не найден! Убедитесь, что вы на странице React приложения'); } else { // Получаем текущее состояние const state = store.getState(); console.log('Текущий пользователь из Redux:', state.auth.user); // Создаем action для обновления пользователя const nonAdminUser = { id: 99999, email: "test.nonadmin@example.com", name: "Тестовый НЕ администратор", is_admin: false, // Ключевой момент - НЕ админ! created_at: new Date().toISOString() }; // Диспатчим action для установки пользователя store.dispatch({ type: 'auth/setUser', payload: nonAdminUser }); // Также обновляем sessionStorage для консистентности sessionStorage.setItem('user', JSON.stringify(nonAdminUser)); console.log('✓ Установлен пользователь БЕЗ прав админа:', nonAdminUser); console.log('Теперь попробуйте перейти на /admin/search'); }

Шаг 4: Альтернативный метод через sessionStorage

Если Redux store недоступен, используйте этот код:

// Устанавливаем пользователя через sessionStorage const testUser = { id: 99999, email: "test.nonadmin@example.com", name: "Тестовый НЕ администратор", is_admin: false, // НЕ администратор created_at: new Date().toISOString() }; sessionStorage.setItem('user', JSON.stringify(testUser)); console.log('✓ Пользователь установлен в sessionStorage:', testUser); // Перезагружаем страницу для применения изменений console.log('Перезагрузка через 2 секунды...'); setTimeout(() => { window.location.reload(); }, 2000);

Шаг 5: Проверка AdminGuard

После установки пользователя выполните в консоли:

// Попытка перейти в админ-панель window.location.href = '/admin/search';

Ожидаемый результат:

  • AdminGuard должен проверить is_admin: false
  • Доступ должен быть заблокирован
  • Должен произойти редирект на главную страницу
  • В консоли появится сообщение об отказе в доступе

Дополнительные команды для отладки

// Проверить текущего пользователя из Redux if (window.__REDUX_STORE__) { const user = window.__REDUX_STORE__.getState().auth.user; console.log('Redux user:', user); console.log('is_admin:', user?.is_admin); } // Проверить пользователя из sessionStorage const sessionUser = JSON.parse(sessionStorage.getItem('user') || '{}'); console.log('SessionStorage user:', sessionUser); console.log('is_admin:', sessionUser.is_admin); // Очистить все данные пользователя sessionStorage.removeItem('user'); if (window.__REDUX_STORE__) { window.__REDUX_STORE__.dispatch({ type: 'auth/logout' }); } console.log('✓ Пользователь удален');
