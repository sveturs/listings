Тест AdminGuard - Обычный пользователь

Описание теста

Этот тест проверяет, что AdminGuard корректно блокирует доступ обычным пользователям (не администраторам) к админ-панели.

Данные пользователя в sessionStorage:

Данные еще не установлены

Инструкция по тестированию:

  1. Нажмите "Установить обычного пользователя" - это создаст пользователя с is_admin: false
  2. Нажмите "Перейти в админ-панель" - откроется новая вкладка с админ-панелью
  3. AdminGuard должен заблокировать доступ и перенаправить на главную страницу
  4. Проверьте консоль браузера (F12) для просмотра логов AdminGuard
