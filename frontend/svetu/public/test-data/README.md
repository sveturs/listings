# Тестовые данные для импорта товаров

## Доступные файлы

### XML формат
- **URL**: `http://localhost:3001/test-data/products-import-sample.xml`
- **Содержит**: 11 тестовых товаров различных категорий

### CSV формат
- **URL**: `http://localhost:3001/test-data/products-import-sample.csv`
- **Содержит**: Те же 11 товаров в CSV формате

## Особенности тестовых данных

- Разные категории товаров (category_id от 1 до 8)
- Различные статусы складских остатков:
  - Обычный запас (8-200 единиц)
  - Низкий запас (3 единицы) - PlayStation 5
  - Нет в наличии (0 единиц) - Samsung Galaxy S22
- Активные и неактивные товары
- Реальные изображения товаров (большинство с CDN производителей)
- Атрибуты товаров в JSON формате
- Цены в EUR

## Использование

1. Перейдите на страницу импорта товаров вашей витрины
2. В поле URL введите один из адресов:
   - `http://localhost:3001/test-data/products-import-sample.xml`
   - `http://localhost:3001/test-data/products-import-sample.csv`
3. Выберите соответствующий формат (XML или CSV)
4. Нажмите "Импортировать"

## Примечания

- Убедитесь, что категории с указанными ID существуют в вашей системе
- Изображения загружаются с внешних источников (кроме olive-oil-bottle.jpg, который должен быть в MinIO)
- Для полного тестирования рекомендуется создать категории с ID от 1 до 8
