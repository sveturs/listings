## Ключевые компоненты UI

### Созданные компоненты:

- `/src/utils/toast.ts` - Менеджер toast уведомлений
- `/src/components/ui/FloatingActionButton.tsx` - FAB кнопка
- `/src/components/ui/SmartForm.tsx` - Умная форма с валидацией
- `/src/components/ui/PageTransition.tsx` - Анимации переходов
- `/src/components/ui/withPageTransition.tsx` - HOC для анимаций
- `/src/components/ui/AnimatedSection.tsx` - Анимированные секции

### Использованные компоненты DaisyUI:

- `skeleton` - для загрузочных состояний
- `btm-nav` - для мобильной навигации
- `modal` - для быстрого просмотра
- `progress` и `radial-progress` - для индикаторов
- `tooltip` - для подсказок
- `card`, `badge`, `alert` - базовые компоненты

## Технологии

- **Next.js 15.3.2** с App Router
- **Tailwind CSS v4** с DaisyUI
- **Framer Motion** для анимаций
- **Heroicons** для иконок
