# Задача выполнена: исправлена кластеризация маркеров на карте

## Статус: Завершено

## Найденная проблема:
Страница map передавала обычные маркеры в компонент InteractiveMap через props `markers={markers}`. 
Это приводило к тому, что компонент показывал обычные маркеры вместо кластеров, даже при zoom < 14.

## Исправления:
1. ✅ Изменен порог кластеризации с 5 на 14 в `InteractiveMap.tsx:113`
2. ✅ Убрана передача `markers={markers}` в `map/page.tsx:679`

## Результат:
- При zoom < 14: показываются синие круглые кластеры с числом объявлений
- При zoom >= 14: показываются индивидуальные маркеры
- Кластеры загружаются динамически через API `/api/v1/gis/clusters`
- При клике на кластер карта приближается

## Проверено:
- Backend API корректно возвращает кластеры
- Frontend перезапущен для применения изменений
- Кластеризация теперь работает правильно
