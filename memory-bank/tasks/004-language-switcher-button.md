# 004 - Изменение размера кнопки переключения языка

## Задача
Кнопка переключения языка была слишком широкой по сравнению с кнопкой "Войти". Необходимо было сделать их одинакового размера.

## Решение
Переделан компонент `LanguageSwitcher` с использованием dropdown menu из daisyUI вместо стандартного select элемента.

### Изменения в коде

**src/components/LanguageSwitcher.tsx**
- Заменен `<select>` элемент на кнопку с dropdown меню
- Использованы классы daisyUI: `dropdown dropdown-end` для контейнера и `btn btn-ghost` для кнопки
- Добавлена стрелка-индикатор для dropdown
- Реализовано выделение активного языка в dropdown меню

### Ключевые моменты
1. Кнопка теперь использует класс `btn btn-ghost`, что делает её такого же размера, как кнопка "Войти"
2. Dropdown меню появляется при клике на кнопку
3. Переключение языка работает корректно - происходит навигация на соответствующую локаль
4. Визуально кнопка отображает текущий язык (RU/EN) с иконкой стрелки

## Результат
- Кнопка переключения языка теперь имеет одинаковый размер с кнопкой "Войти"
- Улучшен UI/UX - dropdown более интуитивен для переключения языка
- Сохранена вся функциональность переключения языков
