# Проблема: Переполнение контекста логами консоли браузера

## Описание
При использовании MCP Playwright для тестирования, консольные логи браузера автоматически собираются и выводятся, что приводит к быстрому переполнению контекста AI.

## Причина
В коде `/home/dim/.npm-global/lib/node_modules/@playwright/mcp/lib/tab.js` на строке 29:
```javascript
page.on('console', event => this._consoleMessages.push(event));
```

## Временное решение
Избегать использования инструмента `browser_console_messages` при тестировании.

## Долгосрочное решение
Необходимо запросить у разработчиков @playwright/mcp добавить опцию для отключения или ограничения сбора консольных логов.
