# Сравнительный анализ логистических партнеров для SVE TU Platform

## Обзор кандидатов

### D Express
**Статус:** Основной кандидат, запрос отправлен (тикет #769179)

**Преимущества:**
- ✅ 250 пакетоматов в 80 городах - идеально для marketplace модели
- ✅ Обработка 65,000+ посылок в день - доказанная масштабируемость
- ✅ Real-time tracking через PDA устройства
- ✅ Современная инфраструктура, ориентированная на e-commerce

**Недостатки:**
- ❌ Нет публичной информации об API
- ❌ Меньше покрытие по сравнению с AKS (80 vs 20 центров)
- ❓ Неизвестна поддержка split billing

### AKS Express
**Статус:** Потенциальный кандидат

**Преимущества:**
- ✅ 20 лет на рынке, устоявшийся бизнес
- ✅ 850 сотрудников, 558 транспортных средств - большие ресурсы
- ✅ Call центр для поддержки
- ✅ Android приложение - есть техническая база
- ✅ Широкое покрытие через 20 центров

**Недостатки:**
- ❌ Нет публичной информации об API
- ❌ С 14.03.2025 наложенный платеж только на банковский счет - усложняет split billing
- ❌ Традиционная курьерская модель, не ориентированная на e-commerce
- ❌ Отсутствие пакетоматов

## Анализ соответствия требованиям SVE TU

| Требование | D Express | AKS Express |
|------------|-----------|-------------|
| API интеграция | ❓ Нужно уточнить | ❓ Нужно уточнить |
| Variable pickup/delivery | ✅ Вероятно (пакетоматы) | ✅ Вероятно |
| Real-time tracking | ✅ Есть через PDA | ⚠️ Только через приложение |
| Split billing | ❓ Нужно уточнить | ❌ Маловероятно (банковские платежи) |
| Покрытие Сербии | ✅ 80 городов | ✅ 20 центров |
| Работа с C2C/B2C | ✅ E-commerce фокус | ⚠️ Традиционная доставка |

## Рекомендация

**Сосредоточиться на D Express и НЕ отправлять запрос AKS Express в данный момент.**

### Обоснование:

1. **D Express - лучшее соответствие модели marketplace:**
   - Пакетоматы критически важны для C2C транзакций
   - E-commerce ориентация соответствует потребностям платформы
   - Масштаб операций (65k посылок/день) подтверждает готовность к росту

2. **AKS Express имеет критические недостатки:**
   - Отсутствие пакетоматов - серьезное ограничение для marketplace
   - Изменение политики наложенных платежей с марта 2025 усложняет split billing
   - Традиционная модель не оптимальна для платформенной экономики

3. **Стратегические соображения:**
   - Уже есть активный диалог с D Express
   - Параллельные переговоры могут размыть фокус
   - Лучше получить конкретные условия от D Express перед рассмотрением альтернатив

## План действий

1. **Немедленно (с D Express):**
   - Дождаться ответа на тикет #769179
   - Подготовить технические требования по API
   - Запросить информацию о split billing возможностях

2. **При необходимости (если D Express не подойдет):**
   - Рассмотреть AKS Express как запасной вариант
   - Изучить другие современные логистические стартапы
   - Оценить возможность работы с несколькими провайдерами

3. **Долгосрочная перспектива:**
   - Планировать multi-carrier стратегию
   - D Express для пакетоматов и городских доставок
   - Традиционные курьеры для специальных случаев
