# Отчет о локализации и региональных особенностях для Сербии

## Сводка

Проект Sve Tu (Све Ту) - это маркетплейс, специально разработанный для сербского рынка с полной поддержкой местных особенностей, включая язык, валюту, географию и культурные аспекты.

## 1. Языковая поддержка

### 1.1 Поддерживаемые языки
- **Сербский (sr)** - основной язык
- **Русский (ru)** - для русскоязычной диаспоры
- **Английский (en)** - международный язык

### 1.2 Особенности сербского языка
- **Полная поддержка кириллицы и латиницы**
  - Система транслитерации между кириллицей и латиницей
  - Специальная обработка диграфов: lj/љ, nj/њ, dž/џ
  - Файл: `backend/pkg/transliteration/comprehensive_test.go`

### 1.3 Названия дней недели на сербском
```json
"days": {
  "monday": "Ponedeljak",
  "tuesday": "Utorak", 
  "wednesday": "Sreda",
  "thursday": "Četvrtak",
  "friday": "Petak",
  "saturday": "Subota",
  "sunday": "Nedelja"
}
```

## 2. Валюта и финансовые аспекты

### 2.1 Основная валюта
- **RSD (Сербский динар)** - основная валюта платформы
- Поддержка других валют: EUR, HRK, MKD

### 2.2 Платежные ограничения
- Минимальная сумма пополнения: 100 RSD
- Максимальная сумма пополнения: 100,000 RSD

### 2.3 Форматирование цен
- Используется стандартный формат с разделителем тысяч
- Пример: "1.000 RSD", "100.000 RSD"

## 3. География и административное деление

### 3.1 Структура адресов
- **Страна**: Сербия (RS)
- **Города**: Полная поддержка всех крупных городов
- **Районы**: Детальное деление для крупных городов

### 3.2 Поддерживаемые города (по приоритету)
1. **Белград/Belgrade** (приоритет: 1)
   - Районы: Вождовац, Врачар, Звездара, Земун, Нови Београд, Палилула, Раковица, Савски венац, Стари Град, Чукарица
   - Пригороды: Барајево, Гроцка, Лазаревац, Младеновац, Обреновац

2. **Нови Сад** (приоритет: 2)
   - Полная поддержка районов

3. **Ниш** (приоритет: 3)
4. **Крагујевац** (приоритет: 4)
5. **Суботица** (приоритет: 5)
6. **Панчево** (приоритет: 6)
7. **Чачак** (приоритет: 7)
8. **Нови Пазар** (приоритет: 8)
9. **Кикинда** (приоритет: 9)
10. **Лесковац** (приоритет: 10)
11. **Смедерево** (приоритет: 11)

### 3.3 Геолокационные особенности
- Использование PostGIS для работы с географическими данными
- Поддержка полигонов для границ городов и районов
- Точные координаты центров населенных пунктов

## 4. Брендинг и маркетинг

### 4.1 Название платформы
- **Sve Tu** (латиница)
- **Све Ту** (кириллица)
- Значение: "Все здесь"

### 4.2 Слоган и описание
- SR: "Sve Tu - tržište za kupovinu i prodaju robe"
- RU: "Sve Tu - торговая площадка для покупки и продажи товаров"
- EN: "Everything Here - marketplace for buying and selling goods"

### 4.3 Целевая аудитория
- Основная: Жители Сербии
- Дополнительная: Русскоязычная диаспора
- Международная: Туристы и экспаты

## 5. Культурные и региональные особенности

### 5.1 Рабочие дни и время
- Рабочая неделя: Понедельник - Пятница
- Выходные: Суббота, Воскресенье
- Поддержка местных праздников

### 5.2 Форматы данных
- Дата: DD.MM.YYYY (европейский формат)
- Время: 24-часовой формат
- Числа: Запятая как разделитель десятичных

### 5.3 Мобильные операторы
Система готова к интеграции с местными операторами:
- Telenor
- MTS
- Yettel (бывший VIP)

## 6. Технические особенности локализации

### 6.1 Файлы локализации
- `frontend/svetu/src/messages/sr.json` - сербские переводы
- `frontend/svetu/src/messages/ru.json` - русские переводы
- `frontend/svetu/src/messages/en.json` - английские переводы

### 6.2 Поддержка мультиязычных адресов
- Миграция 134: `add_multilingual_addresses_support`
- Поддержка адресов на разных языках в БД

### 6.3 SEO оптимизация
- Альтернативные языковые версии: sr-RS, ru-RU, en-US
- Локализованные метатеги для каждого языка

## 7. Интеграции и сервисы

### 7.1 Платежные системы
- Подготовлена инфраструктура для интеграции с местными банками
- Поддержка международных платежных систем (Stripe)
- Mock payment система для тестирования

### 7.2 Геокодирование
- Google Maps API для определения адресов
- Кеширование геокодированных данных
- Поддержка нечеткого поиска по адресам

### 7.3 Уведомления
- Email уведомления
- Подготовка к SMS интеграции
- Telegram бот для уведомлений

## 8. Соответствие местным требованиям

### 8.1 Приватность данных
- Возможность скрытия точного адреса
- Уровни приватности: exact, approximate, hidden
- Соответствие GDPR и местным законам

### 8.2 Налоговая система
- Подготовка к интеграции с налоговой системой Сербии
- Поддержка НДС (PDV)

## Заключение

Проект Sve Tu демонстрирует глубокую локализацию для сербского рынка с учетом языковых, финансовых, географических и культурных особенностей. Платформа готова к запуску на местном рынке и имеет потенциал для дальнейшего развития и интеграции с местными сервисами.
